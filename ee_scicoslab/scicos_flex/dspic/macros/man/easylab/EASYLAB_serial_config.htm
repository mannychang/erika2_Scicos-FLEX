EASYLAB_serial_config - palette: EASYLAB

Description

Palette: EASYLAB

This block controls the UART interface for the EASYLAB Board. The internal parameter specifies the baud-rate.

Parameters

  • Baud-rate: UART baud-rate (9600, 19200, 57600, 115200).
