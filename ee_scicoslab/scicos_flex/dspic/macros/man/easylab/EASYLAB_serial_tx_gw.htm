Scicos block

EASYLAB_serial_tx_gw - palette: EASYLAB

Description

Palette: EASYLAB

This block controls the serial sending of floating point numbers through the UART controller of the dsPIC mounted on the EASYLAB board (the dsPIC UART controller is directly connected with MCP2200, a USB 2.0 to UART converter).

This block accepts two real input values (treated as two single precision floating point numbers).

For PC-side diagrams the 'SERIAL_GATEWAY' block and the UDP blocks, from the 'RT Data Exchange' palette, should be used to estabilish a communication with the board (see the examples for further info).

Parameters

  • Baudrate: The internal parameter specifies the baudrate of the UART controller.
