Scicos block

EASYLAB_leds - palette: EASYLAB

Description

Palette: EASYLAB

This block controls the leds of the Easylab board.

Parameters

  • Led: This parameter specifies the led (1,2,3,4,7,8,9,10).
  • Threshold: This parameter specifies the threshold (from 0.01 to 0.99).
