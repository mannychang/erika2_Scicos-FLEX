Scicos block

EASYLAB_serial_send - palette: EASYLAB

Description

Palette: EASYLAB

This block controls data sending through the serial interface of the Easylab board.

Parameters

  • Baudrate: The internal parameter specifies the UART baudrate (accepted values: 9600, 19200, 57600, 115200).
