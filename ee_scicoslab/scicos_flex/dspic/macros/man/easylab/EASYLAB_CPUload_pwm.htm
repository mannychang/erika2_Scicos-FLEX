Scicos block

EASYLAB_CPUload_pwm - palette: EASYLAB

Description

Palette: EASYLAB

This block generates a PWM signal. The duty cycle is proportional to the CPU load.

Parameters

  • PWM Frequency: This parameter specifies the PWM frequency.
  • PWM pin: This parameter specifies the PWM output pins (from 1 to 4).
