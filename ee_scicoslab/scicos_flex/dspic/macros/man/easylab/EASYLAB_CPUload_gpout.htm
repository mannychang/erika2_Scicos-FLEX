EASYLAB_CPUload_gpout - palette: EASYLAB

Description

Palette: EASYLAB

This block generates a binary signal. The signal remains high during the execution of the diagram so you can measure the calculus time and the CPU load.

Parameters

  • Pin: This parameter specifies the output pin (from 1 to 8).
