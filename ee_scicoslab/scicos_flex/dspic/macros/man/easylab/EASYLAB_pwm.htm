Scicos block

EASYLAB_pwm - palette: EASYLAB

Description

Palette: EASYLAB

This block controls the PWM pins of the Easylab board.

Parameters

  • PWM Frequency: This parameter specifies the PWM frequency.
  • PWM pin: This parameter specifies the PWM output pins (from 1 to 4).
