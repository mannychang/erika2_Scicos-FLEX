EASYLAB_serial_rx_gw - palette: EASYLAB

Description

Palette: EASYLAB

This block controls the serial reception of floats through UART controller for the EASYLAB Board.

Parameters

  • Baudrate: The internal parameter specifies the baudrate of the UART controller.
