FLEX_square - palette: MCHP16-Sources

Description

Palette: MCHP16-Sources

This block generates a square wave signal.

Parameters

  • Amplitude: This parameter specifies the amplitude of the output signal.
  • Frequency: This parameter specifies the frequency of the output signal.
  • Impulse width: This parameter specifies the impulse width of the output signal.
  • Bias: This parameter specifies the bias (vertical offset) of the output signal.
  • Delay: This parameter specifies the initial delay of the output signal.
