Scicos block

UDP_CONFIG - palette: FLEX-PC

Description

Palette: FLEX-PC

This block configures UDP sockets used to send and receive data.

Parameters

  • PC IP address: This parameter specifies PC IP-address.
  • PC port: This parameter specifies PC port.
  • REMOTE IP address: This parameter specifies board IP-address.
  • REMOTE port: This parameter specifies board port.
