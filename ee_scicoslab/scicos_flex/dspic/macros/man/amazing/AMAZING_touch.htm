Scicos block

AMAZING_touch - palette: AMAZING

Description

Palette: AMAZING

This block controls the data-acquisition from the Amazing Ball touch panel.

Parameters

  • Axis: The internal parameter specifies the axis ('X' or 'Y'). The block provides the ball coordinate in signed pixels (example: if the touch panel resolution is 240x180 the coordinate for X will be in the range:-120,119 and for Y:-90,89).
