Scicos block

FLEX_MTB_touch - palette: FLEX-MotionBoard

Description

palette: FLEX-MotionBoard

This block controls a touch panel connected to the FLEX Motion Daughter Board (refer Amazing Ball).

Parameters

  • Axis: This parameter specifies the axis (X,Y).
  • Resolution: This parameter specifies the resolution (from 100 to 4096).
