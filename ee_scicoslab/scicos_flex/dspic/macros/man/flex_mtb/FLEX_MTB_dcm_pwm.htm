FLEX_MTB_dcm_pwm - palette: FLEX-MotionBoard

Description

palette: FLEX-MotionBoard

This block controls the FLEX Motion Daughter Board PWM plugin for DC-Motor. The code of the block assumes a max voltage of 12V and a saturarion voltage of +-6V.

Parameters

  • PWM channel: The internal parameter specifies the channel(from 1 to 2).
