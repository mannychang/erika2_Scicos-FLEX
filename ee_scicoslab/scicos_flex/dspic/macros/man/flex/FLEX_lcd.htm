Scicos block

FLEX_lcd - palette: FLEX

Description

Palette: FLEX

This block controls the LCD which is attached to the first prototype of the Demo Daughter Board (this hardware is no more available, and the block is here for historical usage). If used with the Demo Daughter Board, it controls the LCD of that board.
