Scicos block

FLEX_DMB_lcd_ascii - palette: FLEX-DemoBoard

Description

palette: FLEX-DemoBoard

This block controls the LCD on the Demo Daughter Board. The input ports accept two arrays of digits (up to 16) for the two lines of the LCD.

Usage example: you can link the FLEX_DMB_lcd_ascii block with two constants of value: [ascii('my message'),0]

We suggest to put a zero at the end of the vector because it is the end-string character.
