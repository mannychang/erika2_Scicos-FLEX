Scicos block

FLEX_DMB_buzzer - palette: FLEX-DemoBoard

Description

palette: FLEX-DemoBoard

This block controls the buzzer mounted on the Demo Daughter Board. The input value is the frequency of the buzzer sound, in Hz.
