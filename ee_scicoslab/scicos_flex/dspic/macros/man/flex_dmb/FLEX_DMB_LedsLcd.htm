Scicos block

FLEX_DMB_LedsLcd - palette: FLEX-DemoBoard

Description

palette: FLEX-DemoBoard

This block controls LEDs (8) and LCD of the FLEX DEMO board.

Simulation: This block is fully supported in simulation. During the simulation, an image of the FLEX DEMO board is shown and the status of LEDs and LCD updated. Qt framework has been used to develop this application.

Code Generation: This block is fully supported by the code generator.

Parameters

  • Flex Identifier:

    This parameter can be used to specify an identifier for the board.

    In simulation you can use more than one board.

    This feature is not supported by the code generator.

  • LCD input type:

    This parameter can be used to specify the type of the 2 LCD lines [d: real, u: UINT8].

    For example du (without blank spaces) means that the block accepts:

    - a real number on the first LCD input,

    - an array of UINT8 digits, in ascii format, for the second input.

    If the input type is 'real', the input accepts only a real number, not an array.

    If the input type is 'UINT8', the input accepts an array of UINT8 digits, in ascii format.

    Note: LCD can display only 16 digits for line.
