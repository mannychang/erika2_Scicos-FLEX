FLEX_DMB_acc - palette: FLEX-DemoBoard

Description

palette: FLEX-DemoBoard

This block maps to the accelerometer input of the FLEX Demo Daughter Board.

Parameters

  • Axis: The internal parameter specifies the axis (1=x, 2=y, 3=z).
