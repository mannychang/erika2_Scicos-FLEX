FLEX_DMB_Buttons - palette: FLEX-DemoBoard

Description

palette: FLEX-DemoBoard

This block reads the status of the FLEX DEMO board buttons.

Simulation: This block is fully supported in simulation. During the simulation, an image of the FLEX DEMO board is shown, like the one below. You can press the buttons by clicking with your mouse and the outputs of the block will be updated accordingly. The Qt framework has been used to develop this application.

Code Generation: This block is fully supported by the code generator.

Parameters

  • Flex Identifier:

    This parameter can be used to specify an identifier for the board.

    In simulation you can use more than one board.

    This feature is not supported by the code generator.
