Scicos block

FLEX_DMB_adcin - palette: FLEX-DemoBoard

Description

palette: FLEX-DemoBoard

This block maps to an analog input pin of the FLEX Demo Daughter Board.

Parameters

  • PIN: The internal parameter specifies the ADC pin (from 1 to 8).
