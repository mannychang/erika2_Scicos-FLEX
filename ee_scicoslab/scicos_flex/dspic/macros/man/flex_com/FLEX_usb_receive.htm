FLEX_usb_receive - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block controls the reception of a packet through the USB connection for the FLEX FULL Board.

Parameters

  • ID: The internal parameter specifies the channel ID (from 0 to 14).
