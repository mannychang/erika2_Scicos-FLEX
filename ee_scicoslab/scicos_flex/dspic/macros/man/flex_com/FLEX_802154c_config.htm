Scicos block

FLEX_802154c_config - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block sets the Coordinator parameters for a 802.15.4 based communication for the FLEX Board.

Parameters

  • Local address: Device local address.
  • PAN ID: PAN identifier
  • Radio channel: Zigbee radio channel.
  • Beacon Order: Beacon order (from 3 to 14).
  • Superframe Order: Superframe order (from 3 to 14).
  • GTS device addresses: Vector of GTS addresses (from 0 to 255).
  • GTS lengths: Vector of GTS lengths (from 0 to 2).
  • GTS directions: Vector of GTS directions (0:tx, 1:rx).
