Scicos block

FLEX_802154d_config - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block sets the Device parameters for a 802.15.4 based communication for the FLEX Board.

Parameters

  • Local address: Device local address.
  • PAN ID: PAN identifier
  • Radio channel: Zigbee radio channel.
  • Coordinator address: Address of the coordinator (from 1 to 255).
