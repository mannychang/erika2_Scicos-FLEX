Scicos block

FLEX_serial_send - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block controls the serial sending through the RS-232 protocol for the FLEX Board. The internal parameters specify the serial port and the baud-rate.

Parameters

  • Port: UART port (1,2).
  • Baud-rate: UART baud-rate (9600, 19200, 57600, 115200).
