FLEX_802154_send - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block controls the sending of a packet through the 802.15.4 protocol for the FLEX Board.

Parameters

  • ID: Data ID
  • Address: Destination address
  • GTS flag: Switch flag to enable the GTS.
