FLEX_serial_tx_gw - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block controls the serial sending of floating point numbers through the UART controller of the dsPIC mounted on the FLEX board (this block requires the FLEX daughter RS232 plugin and a suitable USB converter).

This block accepts two real input values (treated as two single precision floating point numbers).

For PC-side diagrams the 'SERIAL_GATEWAY' block and the UDP blocks, from the 'RT Data Exchange' palette, should be used to estabilish a communication with the board.

Parameters

  • Baudrate: The internal parameter specifies the baudrate of the UART controller.
