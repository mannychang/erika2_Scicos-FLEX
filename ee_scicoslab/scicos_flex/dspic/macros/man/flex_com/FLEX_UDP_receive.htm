FLEX_UDP_receive - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block controls the data reception through the UDP protocol for the FLEX Board.

Parameters

  • Data lines: The internal parameter specifies the number of the output lines of the block (from 1 to 15).
