Scicos block

FLEX_canconfig - palette: FLEX-Communication

Description

Palette: FLEX-Communication

This block configures the FLEX CAN driver. The internal parameters specify the message identifiers (only 5 different IDs are supported at the moment).

Parameters

  • ID: CAN message identifier.
