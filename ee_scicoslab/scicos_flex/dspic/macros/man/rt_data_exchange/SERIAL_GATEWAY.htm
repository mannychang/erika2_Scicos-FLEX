SERIAL_GATEWAY - palette: RT Data Exchange

Description

Palette: RT Data Exchange

This block manage external Serial Gateway program, that receive udp packets from scicos PC simulation and send the throught serial channel. Viceversa from the board.

Parameters

  • COM Port: Set wich COM channel the gateway have to use to communicate with the board (1 = COM1, 2 = COM2 3 = COM3 ecc.).
  • Sending Port: Set for the gateway wich port have to use to send UDP packet to scicos simulation.
  • Receiving Port: Set for the gateway wich port have to use to receive UDP packet from scicos simulation.
