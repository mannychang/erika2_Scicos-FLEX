Scicos block

UDP_SENDER - palette: RT Data Exchange

Description

Palette: RT Data Exchange

This block communicate with an microchip MCP2200 chip, to set serial communication parameter and for read and write GPIO.

Parameters

  • Baudrate: Set serial commincation baudrate.

Parameters

  • Input Pins: used to specify the MCP2200 vector of input pins.

Parameters

  • Output Pins: used to specify the MCP2200 vector of output pins.
