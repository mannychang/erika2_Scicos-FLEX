Scicos block

MCP2200 - palette: RT Data Exchange

Description

Palette: RT Data Exchange

This block communicate with an microchip MCP2200 chip, to set serial communication parameters and for read and write GPIO.

Parameters

  • Baudrate: Set serial communication baudrate.
  • Input Pins: used to specify the MCP2200 vector of input pins.
  • Output Pins: used to specify the MCP2200 vector of output pins.
