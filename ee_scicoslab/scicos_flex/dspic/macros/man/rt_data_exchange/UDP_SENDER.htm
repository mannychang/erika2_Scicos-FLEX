UDP_SENDER - palette: RT Data Exchange

Description

Palette: RT Data Exchange

This block send data through a UDP protocol. It can be used also to send data to the FLEX-FULL USB port.

Parameters

  • Data lines: The internal parameter specifies the number of input lines.
